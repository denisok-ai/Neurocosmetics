# Аудит дизайна — HAEE Neurocosmetics Portal

**Дата**: 2025-03-06  
**Цель**: MedTech-стиль, UX/UI, доступность (a11y), консистентность.

---

## 1. Текущее состояние

### 1.1 Палитра и темы
- **Public**: фон `#FAFAFA`, текст `#0B132B`, акцент `#D4AF37` (Champagne Gold) — соответствует референсам.
- **Dashboard**: тёмная тема (Navy `#0B132B`), карточки `rgba(255,255,255,0.05)`.
- **Итог**: палитра выдержана, Medical White + Navy + Gold.

### 1.2 Типографика
- **Шрифты**: Inter (UI), Playfair Display (заголовки) — читаемо, премиально.
- **Проблемы**: нет единой шкалы размеров (часто произвольные text-sm/text-lg); межбуквенный интервал (tracking) не систематизирован для заголовков.

### 1.3 Компоненты и стек
- **Плюсы**: `.glass`, `.glass-gold`, `.hero-product-card`, пробирки (TestTubeChart), анимации (Framer Motion).
- **Проблемы**: кнопки и ссылки без единого `focus-visible` (клавиатурная навигация); часть интерактивных элементов без явного focus-кольца.

### 1.4 UX
- **Плюсы**: чередование секций (bg-white / bg-[#FAFAFA]), логичная иерархия (Hero → Наука → Действие → Протокол → Безопасность → FAQ → Где купить).
- **Проблемы**: нет Skip to content; маркетплейсы в футере ведут на `#`; мобильное меню без aria-expanded/aria-label.

### 1.5 Доступность (a11y)
- **Проблемы**: иконки без aria-label (корзина, кабинет, выход); кнопка бургер-меню без aria-expanded; контраст текста местами на границе (nav text-navy/70).

---

## 2. Рекомендации MedTech / UX

| Область | Рекомендация | Приоритет |
|--------|--------------|-----------|
| **Focus** | Единый focus-visible: ring-2 ring-gold ring-offset-2 ring-offset-background для кнопок и ссылок | Высокий |
| **Секции** | Фиксированная шкала отступов: --section-py (например 6rem/8rem) для консистентности | Средний |
| **Навигация** | Skip link «Перейти к контенту» в начале страницы; aria-label на иконках | Высокий |
| **Футер** | Ссылки маркетплейсов: внешние URL или кнопка «Скоро»; контакт info — кликабельный mailto | Средний |
| **Контраст** | Поднять nav link с navy/70 до navy/80; проверить muted-foreground на WCAG AA | Средний |
| **Кнопки** | Primary = gold, outline = border-navy/15; везде один и тот же размер/радиус по контексту | Низкий |

## 3. План внедрения (скорректированный)

- **D.1** Ввести в globals.css: утилиты focus-visible (MedTech ring), опционально --section-spacing.
- **D.2** Добавить Skip to main content в layout (public).
- **D.3** Navbar: aria-label на иконках, aria-expanded на бургер.
- **D.4** Footer: mailto для email; маркетплейсы — оставить # с title="Скоро" или реальные ссылки.
- **D.5** (Опционально) Типографическая шкала в Project.md и использование в ключевых блоках.

## 4. Ссылки на стандарты

- MedTech/премиум: чистый фон, минимум отвлекающих элементов, доверие через сертификаты и цифры.
